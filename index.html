01 — The Loop

From prompt to deployed, without copy-paste.

ArkOn is the orchestration platform; the loop is the workflow. Every "build me X" trigger walks the same seven steps, in order, with explicit failure modes. Each step owns one skill, one provider, one contract.

Step 01
Research
last30days gathers engagement-scored evidence across Reddit, X, YouTube, HN, GitHub, Polymarket.
Step 02
Moat Pick
visual-moat-database maps niche → foundation reference DNA (Duyu Care, Eight Sleep, Vibrant Wellness).
Step 03
Architect
ArkOn defines the YAML workflow with phases, validation gates, and artifacts.
Step 04
Compress
headroom shrinks the plan payload 60–95% before it hits the LLM. Reversible via CCR.
Step 05
Code
omp (oh-my-pi) executes the plan with MiniMax Coding Plan, native LSP+DAP, hashline edits.
Step 06
Law
sovereign-design-protocol runs the 10-point build checklist. Failure blocks ship.
Step 07
Ship
ArkOn releases via Vercel / GitHub. Decision is imported to gbrain. Loop compounds.
